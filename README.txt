Библиотека для c++builder версия 1.5.7-rc1

Функции для отрисовки любых функций, заданных в коде

LPlotF (х1, х2, у1, у2, колво функций, массив указателей, ссылка на объект Image на необходиомй форме)

Для корректной работы LPlot, необходимо корректно определить функции для отрисовки:

double f1(double x)
{
	return 3*fabs(x)-2*x*x+26;
}

double f2(double x)
{
	return 3*x*x*x*x-8;
}

double f3(double x)
{
	return 3*x+26*x*x;
}

double f4(double x)
{
	return 1/(x*x);
}
//---------------------------------------------------------------------------

void __fastcall TForm1::Button1Click(TObject *Sender)
{
	 const int kf=4;

	 double (*funct[kf])(double) = {f1,f2,f3,f4};

	 LPlotF(-10, 10, -20, 30, kf, funct, Form1->Image1);
}
//---------------------------------------------------------------------------


Задано 4 функции, одна из них имеет точку разрыва в 0 по Х

Так же были заданы границы по Х от -10 до 10 и границы по У от -20 до 30

Был передан массив указателей funct размерности kf, так же kf было передано в функцию отрисовки как параментр кол-во фукнций
